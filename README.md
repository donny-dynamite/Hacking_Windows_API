# Hacking the Windows API with Python
Notes and work based from Brandon Dennis's course - Hacking the Windows API with Python (Udemy)

<br>

Filenames in each folder:
- __barebones_ -> minimal code required to prove concept
- __prod_ -> with comments, constants, structs, function prototypes, context managers, error handling etc

<br>

**Note:** 
- all code written utilising the standard python library, with no third-party modules
- all scripts are stand-alone, where code normally imported (eg via .h header files) is written out in full
- as such, barebones scripts may be 10 lines where the prod equivalent may be in the hundreds... 🫠
